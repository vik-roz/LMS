## Идея проекта
Проект показывает, как можно быстро превратить Excel-табель в рабочий веб-интерфейс для управления учебным процессом.

Вместо передачи файлов и локального запуска:
- система разворачивается как онлайн-демо
- заказчик или куратор может просто открыть ссылку и "покликать"

Это упрощает:
- согласование логики
- проверку интерфейса
- обсуждение до внедрения backend

## Зачем это бизнесу
Проект показывает, как можно быстро превратить Excel-учёт в работающий интерфейс без долгой разработки.

Это полезно, когда:
- процессы уже ведутся в Excel, но неудобны для работы
- нет времени или бюджета на полноценную систему
- нужно быстро показать решение заказчику

Что это даёт:
- наглядную демонстрацию логики (вместо таблиц)
- возможность тестирования без установки
- быстрые итерации и согласование с заказчиком
- основу для будущей полноценной системы

Фактически это переходный этап:
Excel → MVP → полноценная система

## Что реализовано
Преподаватель:
- канбан по модулям
- отображение только непринятых заданий
- карточка студента с проверкой
- принятие / возврат на доработку
- просмотр и открытие сертификата

Студент:
- просмотр прогресса по модулям
- статусы (не начато / на проверке / принято)
- комментарии преподавателя
- доступ к сертификату после завершения курса

## Подход
Проект собран итеративно:
- сначала логика
- затем структура экранов
- затем UX
- затем визуальная доработка

Без попытки сразу сделать «идеально».

## Технически
- index.html — структура
- styles.css — интерфейс
- app.js — логика и данные

Без:
- базы данных
- backend
- сохранения состояния

Все действия работают в памяти браузера.

## Ограничения
- данные не сохраняются после обновления
- нет загрузки файлов
- нет авторизации
